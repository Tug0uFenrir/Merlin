- 参与模型测试

## 🚀 如何开始贡献

### 对于日志贡献者

请遵循 [README.md](README.md) 中的贡献方式。

### 对于代码贡献者

1. **Fork 仓库**

   - 点击 GitHub 页面上的 "Fork" 按钮
2. **克隆您的 Fork**

   ```bash
   git clone https://github.com/your-username/merlin.git
   cd merlin
   ```
3. **创建功能分支**

   ```bash
   git checkout -b feature/your-feature-name
   ```
4. **进行更改**

   - 编写代码
   - 添加测试
   - 更新文档
5. **提交更改**

   ```bash
   git add .
   git commit -m "Add: 简短描述您的更改"
   ```
6. **推送分支**

   ```bash
   git push origin feature/your-feature-name
   ```
7. **创建 Pull Request**

   - 在 GitHub 上创建 PR
   - 描述您的更改
   - 等待审核

## 📝 代码规范

- **语言**：主要使用 Python，支持其他语言如 JavaScript、C++、GO
- **风格**：遵循 PEP 8 (Python)，ESLint (JavaScript)，Google C++ Style Guide (C++)，gofmt (GO)
- **注释**：使用中文注释，保持清晰
- **测试**：为新功能添加单元测试

## 🔍 审核流程

1. **自动化检查**：代码格式、测试通过
2. **人工审核**：至少一位维护者审核
3. **合并**：审核通过后合并
